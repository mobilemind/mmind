Welcome

Welcome to my sandbox. Here you will find utilities for mobile computing and other experiments. Enjoy.

Key links on Mmind.me:

  • Pastelet (generic) - Create a bookmark for Mobile Safari that will paste text&endash; a Pastelet bookmark.
  • Email Pastelet - Create a bookmark for Mobile Safari optimized to paste text into the first Email field on a page.
  • Tel Pastelet - Create a bookmark for Mobile Safari optimized to paste text into the first phone number field on a page.
  • jumpOnJuniper Pastelet - Expedites login process on Juniper “VisitorNet” sites. Very handy on iPads.

Other useful links include:

  • FYI Bookmarklet (on github) - The "fyi" bookmarklets make it easy to select text on a web page and fire off an email that quotes the current selection and includes the page title and URL. All of that is done without you doing copy-and-paste work or back-and-forth app switching.
