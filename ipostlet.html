iPostlet Bookmark Setup

These steps add a bookmark to Mobile Safari that puts the current URL in a new Twittelator post– an “iPostlet” bookmark.

Quick Instructions

Bookmark this page. Edit the bookmark, removing the text before “javascript:”. Save the bookmark.

When you see a page you want to tweet, use the bookmark to open Twittelator with the URL in a new tweet.

Detailed Instructions

Step 1: Add a Bookmark

Tap the add button in the toolbar at the bottom of the screen, then select Add Bookmark.

Take note of the name (or edit it), navigate to a bookmarks folder and then tap Save.

Step 2: Edit the Bookmark

Now you will do an edit to turn the bookmark into the iPostlet for Twittelator.

Tap the Bookmarks button in the toolbar at the bottom of the screen.

Navigate to the folder containing the bookmark you just saved. Do not select the bookmark.

Tap Edit to modify the list and then tap on the bookmark itself.

In the editing panel, tap in the second field (it begins with “http:”). You need edit the beginning of the text, so tap and drag the loupe all the way to the left.

Now place the edit point right after the “?” and delete everything back to the start of the line. When done, the bookmark should look like this:

Step 3: Use the iPostlet Bookmark

The next time you visit a web page you want to post to twitter, open the new bookmark.

Twittelator will open a new tweet that has the web page URL already “pasted” into it.
